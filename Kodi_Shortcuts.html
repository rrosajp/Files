Kodi Shortcuts

 

Windows PC Keyboard

 

Navigate around Kodi with keyboard Arrow keys, Backspace and Enter or with your mouse.

Keep pressing Esc/Back button if you need to get back to Home Screen.

To access "Kodi Settings" select "Quit Kodi" on Home Screen then "Settings" in the menupopup.

To access "Addon Settings" right click mouse or keyboard letter C button on the SELECTED addon.

Subtitles

 

To search alternative subtitles because of out of sync, press Letter M on keyboard or move mouse to bring up "On Screen Controls" then at the bottom select the "Comment Button" next to "Sound Button" to choose alternative subtitles to download, within subtitles page on the left you can search other websites e.g Subscene, Podnapisi and Opensubtitles.

 

Add Video Addons

 

Add more Video Addons go to "Quit Kodi > Settings > Add-ons > Install From Repository > Custom Add-ons Repository > Video Add-ons".

 

Smartphone Remote

 

You can also use your smartphones to remote control Kodi (Both devices need to be on the same WiFi connection).

 

Install Official Kodi Remote App from    or  .

 

1.     Enter the IP Address and Port Number from your main "Home Screen" top left hand corner in to the settings of the app.

2.     Username and password should be blank.

3.     Press Save or Ok to save settings.

4.     Select remote from main menu to control Kodi.
